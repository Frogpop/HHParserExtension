# HHParserExtension
Автоматическое копирование данных с резюме hh.ru в Jira

## Обзор
Расширение позволяет:
1. Автоматически парсить данные с резюме на hh.ru
2. Вставлять данные в поля формы создания/редактирования задач в Jira

## Архитектура
Основные компоненты
### 1. hh_content.js
   **Роль**: Парсинг данных с hh.ru\
   **Функции** :\
   Автоматическое обнаружение резюме\
   Извлечение ФИО, должности и URL\
   Сохранение в chrome.storage.local\
   **Триггер:**\ Автоматически на страницах hh.ru/resume/*
### 2. jira_content.js
   **Роль:** Автозаполнение форм в Jira\
   **Функции:**\
   Обнаружение типа страницы (создание/редактирование задачи)\
   Динамическое обновление полей через MutationObserver\
   Поддержка contenteditable-элементов (описание)\
   **Триггеры:**\
   При открытии popup\
   При нажатии на кнопку "Вставить данные"
### 3. service_worker.js
   **Роль:** Фоновая логика и управление данными\
   **Функции:**\
   Централизованное хранение данных\
   Обработка сообщений между компонентами\
   Очистка storage при выходе из Jira\
   Поддержка кастомных уведомлений
### 4. popup.js
   **Роль:** Интерфейс расширения\
   **Функции:**\
   Запуск автозаполнения\
   Запуск ручной вставки
### 5. manifest.json
   * Разрешения: storage, activeTab, scripting
   * Автоматическая загрузка на hh.ru и Jira
   * Поддержка контент-скриптов и background-сервиса
   * Обработка событий вкладок
